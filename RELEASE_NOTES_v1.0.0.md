# v1.0.0 – Major release (Art Instrument Lab & scaffold milestone)

First major versioned snapshot of francisc.cv. Use this tag as a restore point: you can return to this state anytime with `git checkout v1.0.0` or create a branch from it with `git checkout -b my-branch v1.0.0`.

## Highlights

- **Art Instrument Lab** (`/art`) – Slug-aware hero-art assignment editor for configuring which art appears on work/thinking and hero.
- **Namespaced art assignments** – Art configs support `work:` and `thinking:` namespaces; central scaffold shell vars for layout.
- **Hero canvas** – Algo-v1 hero art pipeline and contour rendering improvements.
- **Scaffold and layout** – Viewport sizing, edge extensions, and color palette updates for accessibility.
- **UI and polish** – Human/machine view switch, SpotlightPanel hover and a11y, README and layout responsiveness.

## Restore to this version

```bash
git checkout v1.0.0
# or, to create a new branch from this point:
git checkout -b restore-v1 v1.0.0
```
